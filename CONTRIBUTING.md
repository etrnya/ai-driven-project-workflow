# Contributing to AI-Driven Fullstack Project Workflow

Thank you for considering contributing to this framework! This project is an evolving standard for AI-assisted software engineering.

## How Can I Contribute?

### 1. Adding New Skills (AI Prompts)
If you have engineered a powerful AI System Prompt or Skill that improves any phase of the SDLC (e.g., a better Database Architect skill, a dedicated GraphQL skill), please submit it!
- Place your skill in the `skills/` directory.
- Ensure it is heavily tested and follows our extreme safety guidelines.

### 2. Enhancing the SOP (README)
If you've encountered a new edge case or architecture pattern in production (e.g., a new AI attack vector), submit a Pull Request to update both `README.md` and `README_zh-TW.md`.

### 3. Reporting Incidents
Did an AI agent do something unexpectedly destructive? We want to know! 
Submit an Issue. Your forensics will help us build better safety guardrails for everyone.

## Pull Request Process
1. Fork the repo and create your branch from `main`.
2. Ensure you update both English and Chinese documentation if applicable.
3. Submit a PR for review.
